# Get started

The Board Configurator app in nRF Connect for Desktop lets you update the configuration of the board controller on some Development Kits from Nordic Semiconductor.
The board controller is the firmware running on the Interface MCU that controls the behavior of the DK.

## Installing the Board Configurator app

For installation instructions, see [Installing nRF Connect for Desktop apps](https://docs.nordicsemi.com/bundle/nrf-connect-desktop/page/installing_apps.html).

## Supported devices

- nRF9161 DK
- nRF9151 DK
- nRF9151 SMA DK
- nRF54H20 DK
- nRF54LM20 DK
- nRF54L15 DK
- nRF54LV10 DK

## Application source code

The code of the application is open source and [available on GitHub](https://github.com/nordicsemi/pc-nrfconnect-board-configurator).
Feel free to fork the repository and clone it for secondary development or feature contributions.
